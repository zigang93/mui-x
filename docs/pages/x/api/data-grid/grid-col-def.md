# GridColDef Interface

<p class="description">Column Definition interface.</p>

## Import

```js
import { GridColDef } from '@mui/x-data-grid-premium';
// or
import { GridColDef } from '@mui/x-data-grid-pro';
// or
import { GridColDef } from '@mui/x-data-grid';
```
